# Bloccapp

App Android per il **blocco selettivo di app** e il monitoraggio della produttività.  
Consente di creare regole personalizzate per limitare distrazioni tramite timer, soglie di utilizzo giornaliero e geofencing.  
Include statistiche d'uso (giornaliere e settimanali), un sistema a punti/livelli gamificato e diversi metodi di sblocco (PIN, QR, biometrico, countdown).  

Tecnologie principali: **Kotlin**, **Jetpack Compose**, **Room**, **Navigation Compose**, **Material 3**, **UsageStatsManager**, **Geofencing API**, **osmdroid**, **ZXing**, **DataStore**.
